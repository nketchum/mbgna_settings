INTRODUCTION
------------

This module contains settings for certain custom modules and the custom
theme for the MBGNA website. This module is a dependency of those modules
and themes.

INSTALLATION
------------

Follow the normal installation procedure for Drupal modules.

CONFIGURATION
-------------

Once the module has been installed, navigate to /admin/config/system/mbgna-settings
(Configuration > System > MBGNA Settings through the administration panel)
to configure the settings.
